# 欢迎来到我的博客

这是本博客系统的第一篇文章。本站基于 **GitHub Pages** 托管，支持以下内容形式：

## 支持的功能

- **文字博客**：使用 Markdown 编写，支持标题、列表、代码块、引用等
- **语音博客**：在文章中嵌入音频播放器
- **视频博客**：在文章中嵌入视频播放器
- **混合内容**：同一篇文章可同时包含文字、音频和视频

## 如何写一篇文章

1. 在 `assets/posts/` 目录下新建一个 `.md` 文件作为正文
2. 在 `js/config.js` 的 `posts` 数组中添加文章信息
3. 根据内容类型设置 `type` 字段：`text` / `audio` / `video` / `mixed`
4. 如有语音或视频，把文件放入 `assets/posts/` 并在配置中填写路径
